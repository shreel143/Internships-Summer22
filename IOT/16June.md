## INTRO TO LOOPS -
- Same code many times 
- Traversing- Searching sorting 
- Two types-
  1. Entry controlled(For , While) 
  2. Exit control(Do While)

### A. FOR LOOP
    ` for (initialisation ; terminal condition ; modification)
      { 
      block of code
      }`
      
 ### B. WHILE LOOP
    ` for (initialisation ; terminal condition ; modification)
      {
